# Cisco Packet Tracer - Faculdade

Este repositório contém dois projetos criados no Cisco Packet Tracer durante a faculdade.

## Projetos

- `primeiro-projeto-cisco.pkt`: o primeiro projeto desenvolvido no Packet Tracer.
- `segundo-proejeto-cisco.pkt`: o segundo projeto desenvolvido no Packet Tracer.

## Descrição

Ambos os arquivos são simulações de redes feitas para atividades acadêmicas, demonstrando topologias, configuração de dispositivos e entendimento prático dos conceitos de redes Cisco.

## Uso

Os projetos podem ser abertos com o Cisco Packet Tracer para visualizar e editar a configuração das redes.
